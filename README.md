# Steam Dashboard

## Overview
This project is an interactive Power BI dashboard built using Steam games data.

The dashboard provides insights into:
- Positive and negative game ratings
- Average and median playtime
- Game pricing analysis
- Genres and categories distribution
- Developers and publishers statistics
- Ownership trends
- Platform comparisons

## Dataset Columns
- AppID
- Name
- Release Date
- Developer
- Publisher
- Platforms
- Required Age
- Categories
- Genres
- Achievements
- Positive Ratings
- Negative Ratings
- Average Playtime
- Median Playtime
- Owners
- Price

## Dashboard Pages

### Page 1
- Game price display
- Game image display
- Platform filter
- Developer filter
- Positive vs Negative ratings comparison

### Page 2
- Ownership analysis
- Achievements analysis
- Required age comparison
- Category distribution
- Top games filters

### Page 3
- Developers by category
- Average price by genre
- Games by required age
- Platform, category and genre KPIs
